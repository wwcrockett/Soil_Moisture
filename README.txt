This folder contains all files needed to execute a Bayes MCMC fit of the SES
half hourly soil moisture and precipitation data with Phydrus, the python
package implementation of HYDRUS-1D modeling software.
All data and HYDRUS read files are contained in the 'read' directory, text filesthat HYDRUS writes to are contained in the 'write' directory, and all processed data is saved to the 'save' directory.
